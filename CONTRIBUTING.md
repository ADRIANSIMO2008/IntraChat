# Contributing to IntraChat

Thank you for your interest in contributing to IntraChat. We welcome contributions that improve the project, fix bugs, or add new features.

Before contributing, please read the guidelines below to ensure a smooth collaboration.

## How to Contribute

1. Fork the repository.
2. Clone your fork to your local machine.
3. Create a new branch for your feature or fix.
4. Make your changes.
5. Test your changes.
6. Commit and push your branch.
7. Open a Pull Request.

## Coding Guidelines

- Follow consistent code style.
- Write clear and descriptive commit messages.
- Keep functions small and readable.
- Add comments where necessary.

## Pull Request Guidelines

Before submitting a Pull Request:

- Ensure the project builds successfully.
- Make sure your code does not break existing functionality.
- Keep pull requests focused on a single change.
- Provide a clear description of what your PR does.
